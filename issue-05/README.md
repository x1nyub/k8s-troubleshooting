# 🚨 트러블슈팅 미션 5: 설정 파일이 마운트되지 않습니다!

## 📝 상황 설명
애플리케이션에 필요한 설정값(`ConfigMap`)을 파일 형태로 파드에 마운트하려고 합니다. 
하지만 `helm install`을 실행하면 파드가 생성되지 않고 설정 마운트 관련 에러가 발생합니다.

## 🎯 목표
1. 에러 메시지를 분석하여 왜 파드 생성이 거부되었는지 파악하세요.
2. 매니페스트 내의 참조 관계를 확인하고 오타를 수정하세요.
3. 파드가 정상 실행된 후, `/etc/config` 경로에 설정값이 잘 들어있는지 확인하세요.

## 🚀 실행 방법
1. **장애 배포:** `helm install mission-5 .`
2. **에러 확인:** 터미널에 출력되는 `Invalid` 에러 메시지를 꼼꼼히 읽어보세요.
3. **해결 후 재배포:** `helm upgrade mission-5 .` (단, 파드는 불변 리소스이므로 에러 시 삭제 후 재배포가 필요할 수 있습니다.)
