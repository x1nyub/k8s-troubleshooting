# 🚨 트러블슈팅 미션 08: 파드가 Pending 상태에서 멈췄습니다!

## 📝 상황 설명
새로운 중요 서비스를 배포하려고 합니다. 그런데 `helm install` 후 파드가 `Running`으로 올라오지 않고 계속 `Pending` 상태에 머물러 있습니다.

## 🎯 목표
1. 파드가 왜 스케줄링되지 않는지 원인을 분석하세요.
2. 현재 노드의 자원 상황과 파드의 요구 사항을 비교하세요.
3. 적절한 리소스 예약값으로 수정하여 배포를 완료하세요.

## 🚀 실행 방법
1. **장애 배포:** `helm install mission-08 .`
2. **상태 확인:** `kubectl get pods` (Pending 확인)
3. **원인 분석:** `kubectl describe pod <파드명>` 명령어의 가장 하단 **Events**를 확인하세요.
