# 🚨 트러블슈팅 미션 1: 끝없는 재시작

## 📝 상황 설명
프로덕션 클러스터에 새 버전의 API 서버(v2.1.0)를 배포했습니다. 
이전 버전은 정상적으로 동작하고 있었으나, 새 버전을 배포하자마자 
파드가 `CrashLoopBackOff` 상태에 빠지며 무한 재시작을 반복하고 있습니다.

## 🎯 목표
1. 파드가 정상적으로 실행되지 않고 계속 종료되는 원인을 찾으세요.
2. Helm 차트 코드를 수정하여 파드를 `Running` 상태로 유지시키세요.

## 🚀 실행 방법
1. **장애 배포:** `helm install mission-1 .`
2. **상태 확인:** `kubectl get pods`
3. **원인 분석 후 수정 및 재배포:** `helm upgrade mission-1 .`
